# 衣搭助手 - 用户使用手册

## 📖 应用简介

衣搭助手是一款智能衣物搭配管理小程序，帮助您轻松管理衣橱，创造完美搭配。通过AI技术，让衣物整理变得简单有趣！

## 🎯 主要功能

### 📸 智能拍照导入
- **相册选择**: 从手机相册中选择衣物照片
- **相机拍摄**: 直接使用相机拍摄新衣物
- **AI抠图**: 自动去除背景，突出衣物主体

### 🎨 拖拽搭配设计
- **直观操作**: 拖拽衣物到画布进行组合
- **实时预览**: 即时查看搭配效果
- **多件搭配**: 支持多件衣物同时搭配

### 💾 自动保存管理
- **1分钟自动保存**: 防止意外丢失
- **云端存储**: 数据安全可靠
- **二次编辑**: 随时修改已保存的搭配

### 🏷️ 智能标签系统
- **自动标签**: AI识别衣物特征
- **自定义标签**: 添加个人化标签
- **快速搜索**: 按标签快速查找

### 🌸 季节主题切换
- **春季主题**: 温暖粉色系 (3-5月)
- **夏季主题**: 清新绿色系 (6-8月)
- **秋季主题**: 温暖黄色系 (9-11月)
- **冬季主题**: 纯净白色系 (12-2月)

### 💡 智能搭配推荐
- **个性化推荐**: 基于您的喜好推荐
- **季节搭配**: 根据当前季节推荐
- **相似搭配**: 基于单件衣物推荐搭配

## 🚀 快速上手

### 第一步：添加衣物

1. **打开衣橱页面**
   - 点击底部"衣橱"标签
   - 点击"+"按钮添加新衣物

2. **选择图片**
   - 选择"从相册选择"或"拍照上传"
   - 选择清晰、光线充足的衣物照片

3. **填写信息**
   - 输入衣物名称
   - 选择分类（上衣、下装、裙子等）
   - 添加标签（颜色、风格、季节等）

4. **保存完成**
   - 点击"保存"按钮
   - AI会自动处理图片背景

### 第二步：创建搭配

1. **选择衣物**
   - 在衣橱页面点击要搭配的衣物
   - 可以选择多件衣物
   - 点击"创建搭配"按钮

2. **拖拽组合**
   - 在画布上拖拽调整位置
   - 调整大小和角度
   - 查看搭配效果

3. **保存搭配**
   - 输入搭配名称
   - 添加场景描述
   - 点击保存

### 第三步：管理标签

1. **查看标签**
   - 在标签页面查看所有标签
   - 按使用频率排序

2. **编辑标签**
   - 点击标签进行编辑
   - 可以修改名称或删除

3. **搜索功能**
   - 使用标签快速搜索衣物
   - 支持多标签组合搜索

## 🎨 使用技巧

### 📸 拍照建议

**最佳拍摄条件:**
- 💡 光线充足，避免阴影
- 🏃‍♀️ 衣物平整摆放
- 🎯 背景简洁纯色
- 📱 镜头稳定对焦

**照片质量要求:**
- 分辨率不低于720p
- 避免模糊和过暗
- 衣物主体完整清晰

### 🏷️ 标签使用技巧

**推荐标签分类:**
- **颜色**: 黑色、白色、红色、蓝色...
- **风格**: 休闲、正式、运动、复古...
- **季节**: 春季、夏季、秋季、冬季...
- **场景**: 职场、约会、旅行、聚会...

**标签命名建议:**
- 简洁明了，3-5个字
- 统一格式，便于搜索
- 避免重复和相似标签

### 👗 搭配建议

**色彩搭配原则:**
- 🎨 同色系搭配
- 🌈 对比色搭配
- ⚫⚪ 中性色调和

**风格搭配技巧:**
- 👔 风格统一性
- 📏 比例协调
- 🎯 场合适宜

## 🔧 常见问题

### Q: AI抠图效果不好怎么办？

A: 
1. 选择背景简洁的照片
2. 确保衣物轮廓清晰
3. 避免复杂背景和阴影
4. 可以重新选择更好的照片

### Q: 为什么无法保存搭配？

A: 
1. 检查网络连接
2. 确认已登录微信账号
3. 检查云存储空间
4. 重新启动应用

### Q: 如何删除不需要的衣物？

A: 
1. 在衣橱页面找到要删除的衣物
2. 点击衣物进入编辑模式
3. 点击删除按钮确认删除
4. 删除后无法恢复，请谨慎操作

### Q: 可以备份我的数据吗？

A: 
1. 所有数据都保存在云端
2. 更换设备时数据会自动同步
3. 可以导出数据到本地
4. 建议定期备份重要搭配

### Q: 如何获得更好的推荐效果？

A: 
1. 完善衣物标签信息
2. 多使用搭配功能
3. 设置个人偏好
4. 根据推荐结果调整设置
