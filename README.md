# 衣搭助手 - 智能衣物搭配管理小程序

👔 一款基于微信小程序的智能衣物搭配管理应用，通过AI技术帮助用户轻松管理衣橱，创造完美搭配。

## ✨ 主要功能

- 📸 **智能拍照导入** - 支持相册选择和相机拍摄，AI自动抠图
- 🎨 **拖拽搭配设计** - 直观的画布操作，实时预览搭配效果
- 💾 **自动保存管理** - 1分钟自动保存，支持二次编辑修改
- 🏷️ **智能标签系统** - AI自动标签 + 自定义标签，快速搜索
- 🌸 **季节主题切换** - 四季主题自动切换（春粉、夏绿、秋黄、冬白）
- 💡 **智能搭配推荐** - 基于用户喜好和季节的个性化推荐

## 🛠️ 技术栈

- **前端**: 微信小程序原生框架 (WXML + WXSS + JavaScript)
- **后端**: 微信云开发 (云函数 + 云数据库 + 云存储)
- **AI服务**: 免费第三方API集成
- **推荐算法**: 基于规则的智能推荐

## 🚀 快速开始

### 环境要求
- 微信开发者工具 v1.06.2301160+
- Node.js v16.0.0+
- 微信小程序账号（已开通云开发）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/clothing-match-app.git
   cd clothing-match-app
   ```

2. **打开微信开发者工具**
   - 导入项目目录
   - 填入AppID
   - 选择云开发模板

3. **配置云开发**
   ```javascript
   // app.js 中配置云环境ID
   wx.cloud.init({
     env: 'your-env-id', // 替换为你的云环境ID
     traceUser: true,
   })
   ```

4. **部署云函数**
   ```bash
   # 在微信开发者工具中右键 cloudfunctions 文件夹
   # 选择"上传并部署：云端安装依赖"
   ```

5. **初始化数据库**
   - 在云开发控制台创建集合：`clothing`、`outfits`、`tags`
   - 设置读写权限

## 🎨 设计特色

### 季节主题系统
- 🌸 **春季**: 温暖粉色系 (#FFB6C1, #FFC0CB)
- 🌿 **夏季**: 清新绿色系 (#98FB98, #90EE90)
- 🍂 **秋季**: 温暖黄色系 (#FFD700, #F0E68C)
- ❄️ **冬季**: 纯净白色系 (#F0F8FF, #FFFFFF)

### UI设计原则
- **卡通风格**: 明亮暖色调，友好亲和
- **圆角设计**: 20rpx-50rpx圆角，柔和视觉
- **微交互**: 悬停效果、加载动画、过渡效果
- **响应式**: 适配不同屏幕尺寸

## 💡 核心算法

### 智能推荐系统
基于多维度用户数据分析：
- **颜色偏好分析**: 统计用户常用颜色
- **风格识别**: 根据标签分析穿衣风格
- **季节适配**: 结合当前季节推荐
- **搭配规则**: 颜色搭配理论 + 风格匹配

### AI抠图技术
- **图像识别**: 自动识别衣物主体
- **边缘优化**: 发丝级边缘处理
- **背景移除**: 智能去背景算法
- **批量处理**: 支持多张图片同时处理
